# video-call-app
Microsoft Teams  Clone Application Built as part of Enagage 2021 Program
App link -https://live-videochat.herokuapp.com/
# Tech Stack:
* HTML
* CSS
* JavaScript
* Nodejs
* Socket.io
* Webrtc
* Peerjs
# Features:
* Chat before and after Meet
* Max 4 Members can interact
* Toggling of Video
* Toggling of Audio
* Live Chat during video call
* invite button
* leave button
